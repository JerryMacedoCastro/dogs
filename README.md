## Iniciando o desenvolvimento

Após clonar o repositório instale as dependências

```bash
yarn 
# ou 
npm install
```

Para rodar o projeto

```bash
yarn start
# ou 
npm start
```

Abra o caminho http://localhost:3000 com o navegador e veja o resultado.
